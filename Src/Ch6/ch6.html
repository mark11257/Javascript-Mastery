CH06: Objects

Modeling Real-World Data

In JavaScript, an Object is a collection of related data and/or functionality. These usually consist of several variables and functions (which are called properties and methods when they are inside objects).

1. Object Creation

You can create objects using the Constructor method (new Object()) or the much more common Literal method ({}).

  • Literal: Fast and easy to read.
  • Constructor: Explicitly tells the engine to reserve space for an object.

Output: Creation

Copy & Try:

const car = { brand: "Tesla", model: "S" };
console.log(car.brand); // "Tesla"

2. Methods & The 'this' Keyword

When you store a function inside an object, it becomes a Method. To access other properties inside the same object, we use the this keyword.

Output: Methods

Copy & Try:

const user = {
  fname: "John Doe",
  greet: function() { return "Hi, " + this.fname; }
};
console.log(user.greet());

3. Iterating with For...In

To look through every "Key" and "Value" inside an object, we use the for...in loop. This is different from the for...of loop used for arrays.

Output: Iteration

Copy & Try (Dynamic Access):

const hero = { name: "John Doe", level: 10, class: "Software Engineer" };

for (let key in hero) {
  // key = "name", "level", etc.
  console.log(key + ": " + hero[key]);
}
