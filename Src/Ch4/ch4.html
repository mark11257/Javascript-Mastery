CH04: Built-in Objects

JavaScript's Standard Library

Every language has a "Standard Library"—a set of pre-built tools. In JavaScript, these are Global Objects like String, Array, and Date. They provide the methods needed to transform data without reinventing the wheel.

1. String Manipulation

In JavaScript, strings are immutable (you can't change the original, only create a new one). Methods allow you to parse and transform text efficiently.

  • Search: Use indexOf() or includes() to find text.
  • Extraction: Use slice(start, end) or substring() to pull specific parts.
  • Transformation: Use replace() with Regex (/pattern/gi) for powerful global search-and-replace logic.

Output: Strings

Copy & Try (Regex & Methods):

const text = "Learn JS, Love JS";
// Replace all 'JS' regardless of case
const clean = text.replace(/js/gi, "JavaScript");
console.log(clean); // "Learn JavaScript, Love JavaScript"

2. Arrays (Collection Management)

Arrays are ordered lists. Unlike other languages, JS arrays can grow dynamically and hold mixed data types (numbers, strings, even other arrays).

  • Stack Methods: push() (add to end) and pop() (remove from end).
  • Queue Methods: unshift() (add to start) and shift() (remove from start).
  • The at() method: Allows using negative indices (e.g., at(-1) gets the last item).

Output: Arrays

Copy & Try (Stack Logic):

const stack = [1, 2];
stack.push(3); // [1, 2, 3]
stack.pop(); // [1, 2]
console.log(stack.at(-1)); // 2

3. Maps & Objects (Key-Value Storage)

Choosing between an Object and a Map depends on your performance and ordering needs.

  • Objects: Best for fixed data structures and JSON records. Keys are mostly strings.
  • Maps: High performance for frequent additions/removals. They preserve order and allow any data type (even objects) as a key.
  • Associative Arrays: In JS, these are just objects. Use a Map if you need to iterate frequently.

Output: Data

Copy & Try (Map Iteration):

const roles = new Map([ ["admin", 1], ["editor", 2] ]);
roles.set("viewer", 3);
for (let [name, id] of roles) { console.log(name, id); }

4. Temporal Logic (The Date Object)

The Date object is the most common "gotcha" for developers because it is zero-indexed for months but not for days.

  • Months: 0 = January, 11 = December.
  • Days: 1-31 (standard), but getDay() returns 0-6 (0 = Sunday).
  • Epoch: Everything is calculated in milliseconds since January 1, 1970.

Output: Time
