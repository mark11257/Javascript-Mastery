### Prerequisites

Before starting, ensure you have:

- **Text Editor**: VS Code (Recommended), Sublime Text, or Brackets
- **Web Browser**: Chrome, Firefox, or Edge (Latest versions)
- **Git**: For version control (optional)

### Installation

1. **Clone the Repository**
   
```
bash
   git clone https://github.com/yourusername/Javascript-Mastery.git
   cd Javascript-Mastery
   
```

2. **Open in VS Code** (Recommended)
   
```
bash
   code .
   
```

3. **Create Your First JavaScript File**
   
```
javascript
   // hello.js
   console.log("Hello, World!");
   
```

### Running JavaScript

You can run JavaScript in multiple ways:

1. **Browser Console**
   - Open Chrome DevTools (F12)
   - Go to Console tab
   - Type your JavaScript code

2. **VS Code (with extension)**
   - Install "Code Runner" extension
   - Press Ctrl+Alt+N to run

3. **Node.js**
   
```
bash
   node filename.js
   
```
